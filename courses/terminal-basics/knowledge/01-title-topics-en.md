# Terminal Basics

## Topics Overview

- What is a terminal / command line?
- Why developers need terminal skills
- Navigating the file system
- Working with files and directories
- Managing processes
- Customizing your terminal experience

## Learning Objectives

By the end of this course, you will be able to:

1. Confidently navigate the file system using terminal commands
2. Create, copy, move, and delete files and directories
3. Understand and manage running processes
4. Combine commands to automate common tasks
5. Customize your terminal prompt and environment

## Prerequisites

No prior experience with the terminal is needed. Just bring your computer and curiosity!
