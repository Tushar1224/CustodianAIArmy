The SQL SELECT Statement
SELECT
statement is used to select data from a database.
Example
Return data from the Customers table:
SELECT CustomerName, City FROM Customers;
Try it Yourself »
SELECT Syntax
SELECT
column1
column2, ...
FROM
table_name
Here,
column1, column2, ...
are the
column names
in the table you want to select data from.
table_name
represents the
name of the table
you want to select data from.
